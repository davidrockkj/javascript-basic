# Comparações

Servem para comparar dois valores e/ou variáveis

- Igual - `==`
- Diferente - `!=`
- Menor - `<`
- Menor igual - `<=`
- Maior - `>`
- Maior igual - `>=`

Uma comparação sempre retornará VERDADEIRO ou FALSO:

```javascript
1 > 2       // false
1 == 1      // true
"A" == "A"  // true
"A" < "B"   // true - ordem do alfabeto
```

Para variáveis

```javascript
const idade = 16
const limite = 18

idade >= limite // false
```
